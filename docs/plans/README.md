# Plans

The campaign layer: P-xxx plans with M-xxx milestones. This folder is
**flat** — status lives in frontmatter, never in folder names. Plans
live on `main`, mutate continuously (bookkeeping in merge digests;
semantic changes only via their own change + ADR), and are **frozen,
never deleted** when their goal is reached: a completed plan is
immutable and this index doubles as the project's achievement overview.
