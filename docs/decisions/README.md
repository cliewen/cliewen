# Decisions

Architecture Decision Records (MADR format) with two-tier provenance:
`inferred` (agent-reconstructed, not yet truth) → `verified` (human has
accepted — the act that makes provenance auditable). Every ADR carries
`author: agent|human` and `accepted-by:`. **Rejected alternatives are
half of "why does the system look like this"** — rejected ADRs live
here too and are never deleted.
